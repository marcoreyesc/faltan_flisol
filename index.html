Publicar en la App Store con Fastlane

En mi infancia, un objeto parecía resumir los remedios para el hombre en apuros: la navaja suiza. "¿Hay vida en la Tierra?" (2012), Juan Villoro

¿Que puedo hacer con Fastlane?

  1. Ejecutar pruebas unitarias
  2. Captura automática de pantallas
  3. Distribución de aplicaciones a TestFlight
  4. Publicar una aplicación en la AppStore
  5. Y mucho pero mucho más!!

https://docs.fastlane.tools/actions/#push

Cómo empezar

$ sudo gem install fastlane -NV

$ fastlane init

Aprovisionamiento

Preferimos gastar el tiempo construyendo cosas, en vez de pelearnos con el aprovisionamento

sigh Nombre corto para la acción get_provisioning_profile

$ fastlane sigh

Gestión de certificados

cert Nombre corto para la acción get_certificates

$ fastlane cert

Trabajando en equipo

match Sincroniza tus certificados y perfiles con todo tu equipo

$ fastlane match

Empaquetar una aplicación

gym Nombre corto para la acción build_ios_app

$ fastlane gym --workspace "Example.xcworkspace" --scheme "AppName" --clean

Enviar una aplicación a pruebas

deliverPara iOS y Android es la manera mas facil de automatizar despligues de un app de pruebas y de liberarla

$ fastlane deliver init

$ fastlane deliver
